# Cyclistic Bike Share Analysis

Interactive Power BI dashboard analyzing Cyclistic bike-sharing data to uncover insights on member vs. casual rider behavior patterns, ride preferences, and usage trends.

## Project Overview
This repository contains a comprehensive Power BI report built from Cyclistic historical bike trip data. The analysis focuses on comparing riding patterns between **annual members** and **casual riders** across key metrics like ride frequency, distance traveled, station usage, and temporal trends.

## Dashboard Breakdown

### 1. Membership vs Casual Rider Usage Dashboard
**Key Features:**
- **KPIs**: Ride ID count, Start station count, End station count
- **Slicer**: Member vs Casual users
- **Visualizations**:
  - Ride Count by Membership Type
  - Member Casual Distribution by Rideable Type
  - Ride Count by User Type and Day Category (Weekend vs. Weekday)
  - Casual Members by Start Day of Week
  - Top 10 Ride Starts by Station

### 2. Distance and Travel Analytics
**Key Features:**
- **Slicers**: Member vs Casual users, Rideable type
- **Multi-row Cards**: Start ride time & distance, End ride time & distance
- **KPI**: Total distance travelled
- **Visualizations**:
  - Total Distance Traveled by Casual Members
  - Total Distance by Starting Station
  - Weekend vs. Weekday Total Distance Traveled
  - Total Distance Traveled by Day of Starting Ride

## Technologies Used
- **Power BI** (DAX, Power Query)
- **Data Analysis**: Ride patterns, station usage, temporal trends
- **Visualizations**: KPIs, slicers, bar/line charts, multi-row cards

## Key Insights Discovered
- Member vs casual ride frequency patterns
- Popular stations and rideable type preferences
- Weekend vs weekday usage trends
- Distance traveled behavioral differences
